About

This is a solver plugin for NGSolve which can be used to solve the compressible Navier-Stokes equations by means of an Hybrid Discontinuous (HDG) Galerkin method. The main interest lies in computational aeroacoustics.

Checkout the documentation (work in progress!) for more information!
